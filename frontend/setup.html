Shopify Connection

Connect your Shopify store

Two options depending on when your app was created:

Option A — Existing custom app (before Jan 2026): You have a static access token. Go to your Shopify Admin → Settings → Apps → your custom app → API credentials → copy the access token. Paste it in the "Static Token" field below.

Option B — New app (Jan 2026+): Static tokens are no longer available. Create a free account at partners.shopify.com → Apps → Create app → select your store → Settings → copy Client ID and Client Secret. Required scopes: write_products + read_products.
