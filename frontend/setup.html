Shopify Connection

Connect your Shopify store

Two options depending on when your app was created:

Option A — Existing custom app (before Jan 2026): You have a static access token. Go to your Shopify Admin → Settings → Apps → your custom app → API credentials → copy the access token. Paste it in the "Static Token" field below.

Option B — New app (Jan 2026+): Static tokens are no longer available. Steps:
1. Create free account at partners.shopify.com
2. Apps → Create app → give it a name
3. Configuration → API scopes: enable write_products + read_products → Save
4. Install app on your store (required — token won't work without this)
5. Settings → copy Client ID and Client secret

Google Ads

Connect your Google Ads account

Eenmalige setup (5 minuten):

1. Ga naar console.cloud.google.com → New project → naam: GoogleClaw
2. Zoek op "Google Ads API" → Enable
3. APIs & Services → OAuth consent screen → External → vul naam + email in → voeg je eigen email toe als testgebruiker → Save
4. Credentials → Create OAuth Client ID → Web application → voeg toe bij Authorized redirect URIs: http://localhost:8080/setup/google-auth/callback
5. Kopieer Client ID + Client Secret hieronder

Developer Token: Alleen beschikbaar in een Manager Account (MCC). Geen MCC? Maak er gratis een aan via ads.google.com/home/tools/manager-accounts, koppel je account, dan: ⚙️ → API Center

Paste your CID here if the account you want is not shown in the list below, or if account discovery returned no results.
